# Matrix Templates

Create prioritization and assessment grids. Choose a template:

## 1. Priority Matrix (Impact/Effort)
**Command:** `/miro-frameworks-matrix-priority`

2×2 matrix (Impact vs Effort). Best for feature prioritization.

**Quadrants:** Quick Wins / Big Bets / Fill Ins / Avoid

**Example:** Feature roadmap, backlog prioritization, initiative planning

---

## 2. Eisenhower Matrix (Urgent/Important)
**Command:** `/miro-frameworks-matrix-eisenhower`

2×2 matrix (Urgent vs Important). Best for task management.

**Quadrants:** Do Now / Plan / Delegate / Drop

**Example:** Time management, daily planning, leadership priorities

---

## 3. Risk Matrix
**Command:** `/miro-frameworks-matrix-risk`

3×3 matrix (Probability × Impact). Best for risk assessment.

**Levels:** Low/Medium/High for both axes

**Example:** Project risks, security risks, business risks

---

Tell the user to choose a template or describe what they want to prioritize/assess.
