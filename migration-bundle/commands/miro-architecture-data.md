# Data Flow Templates

Create data pipelines and relationships. Choose a template:

## 1. Data Flow Diagram
**Command:** `/miro-architecture-data-flow`

Data sources, transformations, and destinations. Best for data processing.

**Example:** ETL flow, data transformation, stream processing

---

## 2. Entity Relationship Diagram (ERD)
**Command:** `/miro-architecture-data-erd`

Database entities and relationships. Best for data modeling.

**Example:** Database schema, data model, relational design

---

## 3. Data Pipeline
**Command:** `/miro-architecture-data-pipeline`

Multi-stage data pipeline with stores. Best for data infrastructure.

**Example:** ETL pipeline, data warehouse flow, analytics pipeline

---

Tell the user to choose a template or describe their data architecture.
