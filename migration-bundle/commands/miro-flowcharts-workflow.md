# Workflow Templates

Create workflow and process automation diagrams. Choose a template:

## 1. Approval Chain
**Command:** `/miro-flowcharts-workflow-approval`

Multi-stage approval process. Best for review/approval workflows.

**Example:** PR approval, expense approval, document review

---

## 2. Pipeline
**Command:** `/miro-flowcharts-workflow-pipeline`

CI/CD style pipeline with parallel stages. Best for build/deploy processes.

**Example:** Build → Test → Deploy pipeline, data processing pipeline

---

## 3. State Machine
**Command:** `/miro-flowcharts-workflow-state`

State transitions and triggers. Best for status/state modeling.

**Example:** Order states, ticket lifecycle, user journey states

---

Tell the user to choose a template or describe their workflow.
