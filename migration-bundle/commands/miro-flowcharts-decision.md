# Decision Tree Templates

Create decision logic diagrams. Choose a template:

## 1. Binary Decision Tree
**Command:** `/miro-flowcharts-decision-binary`

Yes/No decision flow with diamond nodes. Best for boolean logic.

**Example:** User authentication flow, validation checks, if-then logic

---

## 2. Multi-Choice Decision Tree
**Command:** `/miro-flowcharts-decision-multi`

Multiple choice branches (3-5 options). Best for categorical decisions.

**Example:** Payment method selection, user type routing, status handling

---

## 3. Decision Comparison Matrix
**Command:** `/miro-flowcharts-decision-matrix`

Grid comparing options against criteria. Best for structured evaluation.

**Example:** Vendor selection, technology choice, strategic decisions

---

Tell the user to choose a template or describe their decision logic.
