# Command Center

## Description

Command Center is een centraal dashboard voor het beheren van alle Claude Code assets. Het biedt een visueel overzicht van APIs, prompts, skills, agents, commands en instructions die zijn geregistreerd via het reactieve trigger systeem.

## Problems & Solutions

| Problem | Solution |
|---------|----------|
| Assets verspreid over ~/.claude/ mappen | Centraal overzicht in één dashboard |
| Geen zicht op wat er geregistreerd is | Statistieken en lijsten per type |
| Handmatig zoeken in JSON bestanden | Zoeken en filteren in de UI |
| Geen historiek van wijzigingen | Activity log met alle events |

## Key Features

- **Asset overzicht** — Statistieken per type (APIs, Prompts, Skills, etc.)
- **Gedetailleerde lijsten** — Doorbladeren en zoeken per asset type
- **Activity feed** — Chronologisch overzicht van sync events
- **Supabase sync** — Real-time data uit de cloud database
