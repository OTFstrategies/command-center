# Application Shell Specification

## Overview
Minimalist icon-only sidebar navigation with hover-reveal labels. Clean Apple-inspired aesthetic with maximum whitespace and monochrome zinc palette. Subtle blue accents for interactive elements only.

## Navigation Structure
- Home
- APIs
- Prompts
- Skills
- Agents
- Commands
- Instructions
- Activity
- Settings

## User Menu
Location: Top right in minimal header bar
Contents: User avatar, dropdown with user name and logout option

## Layout Pattern
- Narrow icon-only sidebar on the left
- Minimal header bar at the top with user menu
- Large content area with generous padding
- Labels appear on hover via tooltip

## Responsive Behavior
- Desktop: Narrow icon sidebar always visible, header with user menu
- Tablet: Same as desktop
- Mobile: Hamburger icon in header, sidebar slides in as overlay

## Design Notes
- No decorative elements or unnecessary visual noise
- Icons only in sidebar, text labels on hover
- Zinc monochrome palette throughout
- Blue-500 for active states and interactive elements only
- Maximum whitespace in content area
- DM Sans for any visible text, Inter for body content
