# Activity Specification

## Overview
Feed met nieuwe items en gebruik. Filterable per type, periode en project. Infinite scroll voor moderne ervaring.

## User Flows
- Bekijk activiteiten feed chronologisch
- Filter op asset type: APIs, Prompts, Skills, Agents, Commands, Instructions
- Filter op periode: vandaag, week, maand, alles
- Filter op project
- Scroll voor meer activiteiten

## UI Requirements
- Chronologische feed met infinite scroll
- Per item: type icon, asset naam, relatieve tijd, project
- Filters: type dropdown, periode selector, project selector
- Geen paginering
- Geen export functionaliteit
- Relatieve timestamps: vandaag, gisteren, X dagen geleden
