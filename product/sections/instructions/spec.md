# Instructions Specification

## Overview
Workflow en project instructies met scope indicator. Alleen titel in lijst, content in detail view.

## User Flows
- Bekijk lijst van alle instructies
- Filter op scope: workflow of project
- Klik op instructie voor detail view
- Kopieer instructie content

## UI Requirements
- Lijst toont: naam, scope label
- Scope labels: workflow, project als subtiele tags
- Alleen titel in lijst, geen content preview
- Detail view toont volledige content
- Copy functionaliteit

- shell: true
