# Prompts Specification

## Overview
Prompt library met system prompts, project prompts en templates. Registry voor bekijken en kopieren, geen prompt runner.

## User Flows
- Bekijk lijst van alle prompts met type label
- Filter op type: system, project, template
- Klik op prompt voor detail view
- Kopieer volledige prompt content

## UI Requirements
- Lijst toont: naam, type label, preview eerste 2 regels
- Type labels: system, project, template als subtiele monochrome tags
- Detail view toont volledige content
- Variabelen worden getoond maar niet invulbaar
- Copy functionaliteit voor volledige content
