# APIs Specification

## Overview
Beheer van API configuraties met compact lijstoverzicht en uitgebreide detail view. Credentials blijven altijd masked voor veiligheid.

## User Flows
- Bekijk lijst van alle API configuraties
- Klik op API om detail view te openen
- Kopieer API content naar clipboard

## UI Requirements
- Lijst toont: naam, service, auth type
- Compact overzicht, geen endpoints in lijst
- Credentials altijd masked, nooit zichtbaar
- Detail view bij klik met volledige configuratie
- Copy functionaliteit in detail view

## Configuration
- shell: true
