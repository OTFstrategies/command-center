# Settings Specification

## Overview
Supabase sync configuratie met status weergave en per-type selectie.

## User Flows
- Bekijk huidige sync status
- Configureer Supabase connectie
- Selecteer welke asset types gesyncet worden
- Trigger handmatige sync

## UI Requirements
- Sync status weergave: synced, pending, error
- Laatste sync tijd tonen
- Checkboxes per asset type voor sync selectie
- Alle types standaard aangevinkt
- Geen andere settings dan Supabase sync

## Configuration
- shell: true
