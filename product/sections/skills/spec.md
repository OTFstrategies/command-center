# Skills Specification

## Overview
Overzicht van alle skills met bestanden en dependencies. Detail view toont SKILL.md content preview.

## User Flows
- Bekijk lijst van alle skills
- Klik op skill voor detail view
- Bekijk bestanden die bij skill horen
- Bekijk dependencies van een skill
- Kopieer skill content

## UI Requirements
- Lijst toont: naam, beschrijving, aantal bestanden
- Detail view toont: bestandslijst, dependencies lijst
- Dependencies als simpele lijst, geen visualisatie
- SKILL.md content preview in detail view
- Copy functionaliteit
