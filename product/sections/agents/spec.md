# Agents Specification

## Overview
Agent definities met tools en configuraties. Parent relaties simpel weergegeven, geen complexe boomstructuur.

## User Flows
- Bekijk lijst van alle agents
- Klik op agent voor detail view
- Bekijk tools die agent heeft
- Bekijk parent relatie indien aanwezig
- Kopieer agent configuratie

## UI Requirements
- Lijst toont: naam, beschrijving, aantal tools
- Detail view toont: tools lijst, volledige config
- Parent relatie als simpel tekstveld: Parent: naam
- Geen boomstructuur visualisatie
- Copy functionaliteit voor config
