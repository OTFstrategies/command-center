# Commands Specification

## Overview
Slash commands gegroepeerd per categorie met inklapbare subcommands.

## User Flows
- Bekijk commands gegroepeerd per categorie
- Klap categorie open/dicht
- Bekijk subcommands genest onder parent
- Klik op command voor detail view
- Kopieer command content

## UI Requirements
- Groepering per categorie: agent-os, miro, etc
- Subcommands genest onder parent command
- Inklapbare structuur per categorie
- Description getoond per command
- Geen syntax weergave, staat in command file zelf
- Copy functionaliteit
